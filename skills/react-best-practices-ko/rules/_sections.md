# 섹션

이 파일은 모든 섹션, 순서, 영향도 수준 및 설명을 정의합니다.
섹션 ID (괄호 안)는 규칙을 그룹화하는 데 사용되는 파일명 접두사입니다.

---

## 1. 워터폴 제거 (async)

**영향도:** CRITICAL  
**설명:** 워터폴은 성능의 가장 큰 적입니다. 순차적인 await는 매번 전체 네트워크 지연 시간을 추가합니다. 이를 제거하면 가장 큰 성능 향상을 얻을 수 있습니다.

## 2. 번들 크기 최적화 (bundle)

**영향도:** CRITICAL  
**설명:** 초기 번들 크기를 줄이면 TTI(Time to Interactive)와 LCP(Largest Contentful Paint)가 개선됩니다.

## 3. 서버 사이드 성능 (server)

**영향도:** HIGH  
**설명:** 서버 사이드 렌더링과 데이터 패칭을 최적화하면 서버 측 워터폴이 제거되고 응답 시간이 줄어듭니다.

## 4. 클라이언트 사이드 데이터 패칭 (client)

**영향도:** MEDIUM-HIGH  
**설명:** 자동 중복 제거 및 효율적인 데이터 패칭 패턴으로 불필요한 네트워크 요청을 줄입니다.

## 5. 리렌더링 최적화 (rerender)

**영향도:** MEDIUM  
**설명:** 불필요한 리렌더링을 줄이면 낭비되는 연산을 최소화하고 UI 반응성을 개선합니다.

## 6. 렌더링 성능 (rendering)

**영향도:** MEDIUM  
**설명:** 렌더링 프로세스를 최적화하면 브라우저가 수행해야 할 작업이 줄어듭니다.

## 7. 자바스크립트 성능 (js)

**영향도:** LOW-MEDIUM  
**설명:** 핫 패스에 대한 마이크로 최적화가 누적되어 의미 있는 개선으로 이어질 수 있습니다.

## 8. 고급 패턴 (advanced)

**영향도:** LOW  
**설명:** 신중한 구현이 필요한 특정 케이스를 위한 고급 패턴입니다.
